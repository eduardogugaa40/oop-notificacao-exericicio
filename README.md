Este repositório contém um exercício de estudo sobre Programação Orientada a Objetos em Java, com foco nos conceitos de interfaces, polimorfismo e desacoplamento. O objetivo é praticar a definição de contratos por meio de interfaces e a utilização de diferentes implementações de forma genérica, sem acoplamento direto às classes concretas. Trata-se de um projeto educacional, voltado exclusivamente para aprendizado e experimentação dos conceitos.
